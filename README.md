# Usman_Iqbal_Portfolio
Modern portfolio built with React.js showcasing my projects, skills, and experience in AI, ML, and Full Stack Development.

This is my modern and fully responsive personal portfolio website built using React.js and JavaScript, designed to showcase my skills, projects, and experience in an interactive and visually engaging way.

The portfolio features a clean UI, smooth animations, and a structured layout that highlights my work in AI, Machine Learning, Web Development, and Salesforce automation.

✨ Key Features:
- Interactive and modern UI/UX design
- Fully responsive across all devices
- Smooth animations and transitions
- Project showcase with detailed descriptions
- Skills and experience section
- Contact integration

🛠️ Tech Stack:
- React.js
- JavaScript (ES6+)
- Tailwind CSS
- Three.js (for 3D elements)
- HTML5 & CSS3

📂 Sections Included:
- Hero Section (Introduction & Overview)
- About Me (Background & Skills)
- Experience (Professional & Academic Work)
- Projects (Highlighted Work)
- Contact Section (Connect with me)

🎯 Purpose:
This portfolio is designed to represent my professional identity, demonstrate my technical skills, and provide a platform for recruiters and clients to explore my work.

💡 About Me:
I am a final-year Computer Science student with a strong interest in AI, Machine Learning, and Full Stack Development. I enjoy building intelligent systems and solving real-world problems through technology.
